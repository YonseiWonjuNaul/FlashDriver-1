If you want to use BM(Block Manager), have to do following lists.

1. blockmanager/ 디렉토리가 제대로 FlashSimulator 안에 들어있는지 확인한다.

2. ftl에서 c파일들이 모두 blockmanager의 자료구조를 사용할 수 있도록 header에 다음의 코드를 추가한다.
	#include "../../blockmanager/BM_Interface.h"  // blockmanager 디렉토리의 BM_Interface.h 를 include하면 된다.
	extern Block* blockArray;
	extern Block** numValid_map;
	extern Block** PE_map;

3. FlashSimualtor의 Makefile에서 78행의 libsimulator.a 부분을 다음과 같이 수정한다(blockmanager 디렉토리 추가). blockmanager 디렉토리의 'Makefile_changed.png'를 참고하면 된다.
	cd ./blockmanager && $(MAKE) && cd ../

4. 이제, ftl 코드 내에서 BM의 API를 사용할 수 있다. BM_Interface.c에 정의된 함수들을 사용하면 된다.
